Prefix CSS

- Khái quát: CSS có rất nhiều thuộc tính nhưng không phải thuộc tính nào cũng hỗ trợ tất cả trình duyệt.
- Khi gặp các thuộc tính không hỗ trợ all trình duyệt cần thêm tiền tố prefix để được hỗ trợ.

- Một số tiền tố ứng với các trình duyệt:

  • Internet Explorer: -ms-
  • Chrome: -webkit-
  • Safari: -webkit-
  • Ios: -webkit-
  • Android: -webkit-
  • Firefox: -moz-
  • Opera: -o-
  • Microsoft edge: -ms-
- Caniuse.com: Trang web hỗ trợ kiểm tra thuộc tính CSS có cần sử dụng thêm prefix để hỗ trợ đa trình. (caniuse.com)
- Autoprefixer CSS online: Trang web thêm prefix cho những thuộc tính cần thiết. (https://autoprefixer.github.io/)
