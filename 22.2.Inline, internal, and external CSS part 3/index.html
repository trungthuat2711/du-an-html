Tổng quan CSS(Cascading Style Sheets)

- Là 1 ngôn ngữ dùng để tìm và định dạng lại các phần tử được tạo ra bởi ngôn ngữ đánh dấu HTML
- Nếu HTML đóng vai trò định dạng các phần tử trên website như việc tạo ra các đoạn văn bản các tiêu đề,...thì CSS sẽ giúp thêm style vào các phần tử HTML đó như thay đổi bố cục, màu sắc trang,đổi màu chữ,font chữ,...

Các vị trí đặt CSS

- Có 3 nơi có thể đặt CSS:

  1. Inline (CSS nội tuyến).
  2. Internal (CSS nội bộ).
  3. External (CSS bên ngoài).

- Trong trường hợp muốn áp dụng cho toàn bộ các file html nằm trên trang web, tránh việc phải copy CSS ra tất cả các file này.
- Là cách viết mã CSS thành 1 tệp .css quản lý riêng.
- Cần khai báo trong HTML để link đến tệp CSS: <link rel="stylesheet" href="style.css"> (đặt trong phần head).
- rel="stylesheet" thông báo cho trình duyệt tải tệp CSS được chỉ định và áp dụng các định dạng kiểu trong tệp đó cho trang web hiện tại.
