HTML code tag - Tìm hiểu và sử dụng thẻ code để định dạng mã nguồn trong trang web

Khái niệm

Thẻ <code>: là 1 thẻ có ngữ nghĩa, sử dụng khi có 1 đoạn code trình duyệt sẽ hiểu đây là 1 đoạn code.
Thể hiện 1 khối code độc lập(cần chuyển các kí tự đặc biệt trong khối code sang HTML Entities).
Lưu ý: không thụt thẻ code vào trong, dùng kèm thẻ pre để giữ định dạng khối (https://codebeautify.org/html-escape-unescape).

Ví dụ

    <head>
      <meta charset="UTF-8" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <title>
        HTML code tag - Tìm hiểu và sử dụng thẻ code để định dạng mã nguồn trong
        trang web
      </title>
    </head>
